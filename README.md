# everyday-tasks

Everyday Tasks - це консольний застосунок для керування списком щоденних завдань.
Програма дозволяє додавати, переглядати, редагувати та видаляти завдання,
а також позначати їх як виконані або невиконані.
Список завдань зберігається у локальному файлі.

## Функціональні можливості
- додавання нових завдань
- перегляд списку завдань
- редагування назв завдань
- позначення завдань як виконаних або невиконаних
- видалення завдань
- збереження списку у файл

## Технології
- мова програмування: C++
- тип застосунку: консольний (CLI)

## Запуск проєкту

Програма — консольна (C++). Дані зберігаються у файлі `tasks.txt` у папці проєкту.

### Варіант 1: CLion (Windows/macOS/Linux)
1. Відкрити проєкт у CLion
2. Обрати конфігурацію `app` (або запуск `main.cpp`)
3. Натиснути Run ▶️

### Варіант 2: Windows (Visual Studio)
1. Відкрити Visual Studio → Create a new project
2. Обрати Console App (C++)
3. Додати у проєкт файл з кодом (наприклад `main.cpp`) або замінити вміст стандартного файлу
4. Натиснути Local Windows Debugger / Start

### Варіант 3: Командний рядок (g++)
**macOS / Linux:**
```bash
g++ -std=c++17 main.cpp -o app
./app
