# User Manual — Everyday Tasks

## 1. Призначення програми
Everyday Tasks — консольна ToDo-програма для ведення списку щоденних завдань.
Вона дозволяє додавати завдання, переглядати список, редагувати, видаляти та позначати виконання.
Дані зберігаються локально у файлі `tasks.txt`, тому після перезапуску програми список не втрачається.

## 2. Запуск програми
Після запуску відкривається головне меню. Користувач вводить номер пункту меню і натискає Enter.

## 3. Опис меню та функцій

### 1) Додати завдання
- Оберіть пункт **1**
- Введіть назву завдання (рядок тексту) і натисніть Enter  
Результат: завдання додається у список зі статусом **невиконано**.

### 2) Переглянути список
- Оберіть пункт **2**  
Результат: програма показує всі завдання з нумерацією та статусом:
- `[ ]` — невиконано
- `[x]` — виконано  
Після перегляду натисніть Enter, щоб повернутися до меню.

### 3) Позначити виконане/невиконане
- Оберіть пункт **3**
- Програма покаже список завдань
- Введіть номер завдання  
Результат: статус завдання зміниться на протилежний:
- невиконано → виконано
- виконано → невиконано

### 4) Редагувати назву завдання
- Оберіть пункт **4**
- Програма покаже список завдань
- Введіть номер завдання
- Введіть нову назву  
Результат: назва обраного завдання буде змінена.

### 5) Видалити завдання
- Оберіть пункт **5**
- Програма покаже список завдань
- Введіть номер завдання  
Результат: обране завдання буде видалено зі списку.

### 6) Зберегти у файл
- Оберіть пункт **6**  
Результат: поточний список завдань буде збережено у файл `tasks.txt`.

### 0) Вийти
- Оберіть пункт **0**  
Результат: програма завершує роботу. Перед виходом список завдань автоматично зберігається у файл.

## 4. Збереження даних
- Файл `tasks.txt` створюється у папці проєкту.
- Під час запуску програма автоматично намагається завантажити завдання з `tasks.txt`.
- Якщо файлу немає, програма стартує з порожнім списком.

## 5. Примітки
- Назва завдання не може бути порожньою.
- Номер завдання потрібно вводити в межах показаного списку (від 1 до кількості завдань).
